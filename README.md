# Smart EVA Suit Bio-Monitoring and Early Warning System

## Overview
A software-based prototype that simulates astronaut health monitoring during Extravehicular Activities (EVA). The system analyzes sensor data and generates early warning alerts.

## Features
- EVA suit sensor data simulation
- Health risk analysis
- Early warning alerts
- Software-only solution

## Tech Stack
- Python

## How to Run
Run the file `alert_system.py` using Python.

## Note
This project is a conceptual prototype created for hackathon demonstration.
